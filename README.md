# RoboCore
RoboCore is an AI bot that combines using advanced conversational intelligence with real world physical interaction. 

Unlike traditional chatbots, RoboCore integrates GEMINI API with voice recognition, speech synthesis, robotic movements and hardware control systems.

This bot can understand spoken commands, process it, produce a response, navigate its environment and perform physical tasks.

Key Features
Real-time voice-to-voice conversations
Gemini-powered reasoning and decision making
Autonomous movement and navigation
Physical task execution through hardware integration
Natural speech generation and audio responses
Continuous speech recognition and command processing

RoboCore aims to bridge the gap between digital intelligence and physical robotics, creating an assistant that not only understands and communicates like an AI companion but can also interact with and assist in the real world.

This project explores the future of embodied artificial intelligence, where advanced language models are combined with autonomous robotic systems to create capable, interactive, and intelligent machines.

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1lsPJCjeYTjUVX-dKiMkDSYhjVJUqmR47

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
